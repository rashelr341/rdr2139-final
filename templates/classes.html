Classes I've Taken This Semester


  1. Data Structures in Java

    This has probably been my favorite class of all this semester. Not only is Blaer an amazing professor full of humorous stories, but his ability to answer any question can make any lecture easy to follow. After Blaer's class, I am confident that I can apply my new skills to future projects.

  2. Physics: Introduction to Electricity, Magnetism, and Optics

    Although physics isn't my forte, I did enjoy this class. I'm just glad my major doesn't require another physics course, unlike a few of my MechE friends.

  3. Introduction to Computing for Engineers and Applied Scientists

    This is probably my second favorite class. Java will always have a place in my heart, I'm sorry Python. I do have to say that although I had some coding experience prior to this class, I found this to be one of the most difficult intro to CS classes I've ever taken. Nevertheless, I don't regret it because it brought to the table something that the other classes didn't: real-life applications to the world. My favorite assignment has to be the library we built to determine if a tumor was benign or malignant based on a database of known tumors.

  4. Introduction to Experimental Physics

    This was a physics lab class where we learned how to write a college-level lab report and perform various experiments in a lab environment. Not my favorite class, but I did learn several skills that I will apply when writing my thesis in graduate school.

  5. Physical Education: Strength Training

    Forced to take a PE course, I dreaded the day I'd have to walk into the college gym and (gasp) exercise. However, I soon learned that I liked strength training! I've learned many muscle-building routines that I continue to implement today. The one regret I do have about this class is that I never tried to bench press. Hopefully, after quarantine is over and we can return to campus, I can complete this goal of mine.
